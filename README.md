## 列表

### 推荐

1. ![风格1](./imgs/table-1.png)

* 上部分查询条件可以根据需要展开和收起

* 使用开放式表格（没有太多边框），透气性好

* 操作使用文字时，用”|“分隔，并保持间距

  

2. ![风格2](./imgs/table-2.png)

* 如果要展示的列特别多，可以使用不同的前置图标或者颜色，把内容分类展示

### 避免

![](./imgs/table-3.png)

* 高度太紧促
* 排序图标、删除图标比较突兀
* 行和列不知所云
